# Digital Business Card

デジタル名刺のWebページプロジェクトです。

## 概要
このプロジェクトは、モダンで美しいデジタル名刺を作成するためのWebアプリケーションです。

## 技術スタック
- HTML5
- CSS3
- JavaScript
- レスポンシブデザイン

## 開発環境のセットアップ
```bash
git clone https://github.com/KMiya84377/digital-business-card-dev.git
cd digital-business-card-dev
```

## ブランチ戦略
- `main`: 本番用のブランチ
- `dev`: 開発用のブランチ
